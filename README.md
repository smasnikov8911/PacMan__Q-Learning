# PacMan__Q-Learning
Описание:
Это проект по игре PacMan, где персонаж учится играть сам с помощью Q-learning. За основу я взял готовый проект https://github.com/usawa/pypacman и переделал его под обучение с подкреплением. Делал всё в учебных целях.

Стек:
Python, Pygame, NumPy, Jupyter Notebook, Q-learning

Как это работает:
При первом запуске создаётся Q-таблица, в которой Пакман постепенно запоминает, какие действия лучше делать в разных ситуациях. После обучения таблица сохраняется в файл q_table_with_ghost.pkl.

Файл themes:
В нём находятся текстуры для игры. Часть из них я делал вручную.

Как проходит обучение:
Пакман смотрит на ситуацию вокруг, выбирает действие, получает награду или штраф и на основе этого обновляет Q-таблицу. Например, за съеденную точку он получает плюс, за приближение к призраку — минус, а если его поймали, то большой штраф.
